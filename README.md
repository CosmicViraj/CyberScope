**CyberScope** is a Python Tkinter-based SOC-style cybersecurity assessment tool. Designed for **ethical security testing**, labs, and academic projects, it integrates Nmap scans, offline CVE lookup, risk scoring, visual graphs, scan history, and PDF reports.

## Features
- SOC-style **Tkinter desktop dashboard**
- **Nmap integration** (authorized scans only)
- **Offline CVE lookup**
- **Risk scoring** (Low / Medium / High)
- **Graphs**: Ports vs Risk
- **Scan history tracking**
- **PDF report generation**
- Tested in **Kali Linux & Metasploitable labs**

## Technologies
- Python
- Tkinter
- Nmap
- Matplotlib
- ReportLab
- JSON
